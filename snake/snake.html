Installer


Appuyez sur le bouton "Installer", vous allez avoir un fichier zip avec 3 fichiers java. Pour la suite il vous faudra utiliser un interpréteur comme VSCode ou BlueJ.


Pour lancer le fichier il vous faudra lancer la méthode "main" du fichier "Snake" Vous allez pouvoir jouer au jeu


Si vous ouvrez le programme "gamePanel", a la ligne 16 vous aurez la variable "delay", vous pouvez augmenter ou décrémenter sa valeur afin de modifier la vitesse de jeu.

Esperant que ça vous plaise
